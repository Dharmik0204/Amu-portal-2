# 🌱 AgriHealth Portal (MERN Stack)

A full-stack web application for digital farm monitoring and healthcare management. This platform connects farmers, veterinarians, medical stores, and administrators in one system.

---

## 🚀 Live Demo
Frontend: https://your-frontend.vercel.app  
Backend API: https://amu-portal-2.onrender.com  

---

## 📌 Features

- 🔐 Authentication & Authorization (JWT-based)
- 👨‍🌾 Farmer Dashboard
- 🩺 Veterinarian Panel
- 💊 Medical Store Module
- 🧑‍💼 Admin Control Panel
- 📱 Responsive UI Design
- 🔄 REST API Integration

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Deployment:**
- Vercel (Frontend)
- Render (Backend)
